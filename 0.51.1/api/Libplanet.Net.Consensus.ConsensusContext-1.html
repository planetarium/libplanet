HandleMessage(ConsensusMsg)

Handles a received ConsensusMsg by either dispatching it to the right Context<T> or discarding it.

In particular, this discards consensusMessage with Height less than Height. Otherwise, given consensusMessage is passed on to a Context<T> with Height the same as Height of consensusMessage. If there is no such Context<T>, then a new Context<T> is created for the dispatch.
