Type Parameters

T

An IAction type.
Remarks

Due to a limitation in the current version (v2.0.3) of Libplanet.Explorer.Cocona, the generic class cannot be used as follows:

[HasSubCommands(typeof(IndexCommand<PolymorphicAction<ActionBase>>), "index")]

Instead, you would have to subclass this class with the desired IAction type and use the new class with Libplanet.Explorer.Cocona:

public class IndexCommand : IndexCommand<PolymorphicAction<ActionBase>>;
{
}
