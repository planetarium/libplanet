Class ConsensusContext

A class that maintains a collection of Contexts.

Internally, multiple Contexts may be managed at any given time. In particular, a Context with its Height that is greater than Height may be created preemptively to hold ConsensusMsgs to be processed at later time. Furthermore, it is guaranteed that no Context with its Height less than Height, as any such Context is disposed when Height is updated.

HandleMessage(ConsensusMsg)

Handles a received ConsensusMsg by either dispatching it to the right Context or discarding it.

In particular, this discards consensusMessage with Height less than Height. Otherwise, given consensusMessage is passed on to a Context with Height the same as Height of consensusMessage. If there is no such Context, then a new Context is created for the dispatch.
