Class RemoteKeyValueStore

The gRPC client for IKeyValueStore. This class is a thin wrapper around the KeyValueStore.KeyValueStoreClient generated by protoc.

RemoteKeyValueStore needs to be used with the RemoteKeyValueService. The RemoteKeyValueService should be hosted on a remote server or a local process, and the RemoteKeyValueStore should be used on a client.
