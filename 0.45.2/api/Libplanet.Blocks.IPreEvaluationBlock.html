Interface IPreEvaluationBlock

A common interface for blocks that have their proof-of-work Nonces and Transactions, but are not evaluated yet (i.e., their state root hashes are not yet determined).

Remarks

It is similar with IPreEvaluationBlock<T> but it is non-generic interface. It means that it doesn't check the action types in serialization.
