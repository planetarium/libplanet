Class MockWorldState

A mock implementation of IWorldState with various methods for setting up a semi-valid IWorldState for testing.

For ease of use, any mutating method automatically commits the resulting MockWorldState to the IStateStore that the instance is tied to.

If simply an empty disposable IWorldState is needed, use MockLegacyWorldState or MockModernWorldState instead.
