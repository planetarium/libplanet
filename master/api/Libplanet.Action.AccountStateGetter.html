Delegate AccountStateGetter

An delegate to provide read-only view of account states.

Gets an account state of the given address.

If the given address has never been set its account status, returns null instead of throwing any exception.

Namespace: Libplanet.Action
Assembly: Libplanet.dll
Syntax
